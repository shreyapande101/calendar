# CALENDAR

##  About Project 

This is the simple WEB-CALENDAR. Made for improving skills in coding
it have very user friendly ui.In this calendar user can check any date from past,present and future. It also contain clock which shows current time and also current Date too.

### Language Used
- **HTML** – for structure
- **CSS** – Styling, layout, colors.
- **JavaScript** – for functionality.
